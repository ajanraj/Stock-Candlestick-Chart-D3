Introduction

This project is a data visualisation project that uses D3.js to create visualisations of stock data. The data is sourced from Yahoo Finance and is in the form of CSV files. The visualisations include a CandleStick Chart, a Line Chart and a Comparison Chart.

These visualisations serve to answer Research Question 3 of part 3 of the Data Visualisation coursework

  • Analyse the price movements of the stocks of various companies?

    • Providing data for this research question - there are three visualistions in total clickcable under the left-side navbar:

  • A CandleStick Chart
  • A Line Chart
  • Lastly, a Comparision Chart

    • Key Information

    Development of the visualizations was multi-faceted, incorporating the following technologies and features:

    • HTML and JavaScript for structure and functionality
    • D3.js v.7 for dynamic data visualization, available at https://d3js.org
    • Interactive elements such as responsiveness on hover and zoom
    • Custom CSS for improved user experience

    As an open-source project, each visualization builds upon the work of other developers, detailed as follows:

    Candlestick Chart

    This chart is based on code developed by the D3 community, available at https://observablehq.com/@d3/candlestick-chart/2.

    Line Chart

    This chart is based on code developed by the D3 community, available at https://observablehq.com/@d3/line-chart/2.

    Comparison Chart

    This chart is based on code developed by the D3 community, available at https://observablehq.com/@d3/line-chart/2.

    User Interface

    The UI was inspired by the work of Humphrey Curtis, with the original project available at https://github.com/HumphreyCurtis/7CCSMSDV-Project-Data-Visualisation-2/.

    About the data

    The data is sourced from Yahoo Finance and is in the form of CSV files. The data includes the stock prices of various companies. The data is in the form of a CSV file and includes the following columns: Date, Open, High, Low, Close, Change, Change % and Volume.
